Div is a block element

Pointers For Div

  1. width of block level elements is 100% of the viewport width
  2. Height of the block is not set by default
  3. Height and width can be explicitly set by CSS

Span is inline element.

Pointers For span inline (element)

  1. width of inline level element is dependent on the content
  2. Height of the inline is not set by default
  3. Height and width can not be explicitly set by CSS
